Comment utiliser STL Sentinel

1

Configurez les réglages

Définissez les dimensions maximales de votre plateau d'impression et ajustez le barème de notation pour chaque critère.

2

Importez vos fichiers STL ou OBJ

Glissez-déposez ou sélectionnez un ou plusieurs fichiers STL / OBJ. L'analyse se lance automatiquement.

3

Consultez les résultats

Chaque fichier est analysé : dimensions, volume, nombre de triangles, manifold… Les corrections nécessaires sont listées avec le score obtenu.

4

Exportez le rapport

Téléchargez un récapitulatif CSV de tous les fichiers analysés avec leurs scores et corrections.
